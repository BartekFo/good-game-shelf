# Docs

Ten katalog zawiera uproszczona dokumentacje dla projektu `Game Explorer`.

## Zakres dokumentacji

- `01-concept/game-explorer-concept.md` - pomysl na aplikacje i ekrany,
- `02-requirements/requirements.md` - wymagania funkcjonalne i niefunkcjonalne,
- `03-architecture/flutter-architecture.md` - prosta architektura Flutter bez wlasnego backendu,
- `04-product-strategy/positioning-and-naming.md` - pozycjonowanie produktu, MVP i zasady bezpiecznego nazewnictwa,
- `05-delivery/mvp-user-stories-and-cycles.md` - user stories MVP i rozbicie na issues oraz cykle pracy,
- `05-delivery/cycle-1/` - folder z planem pierwszego cyklu MVP i osobnymi katalogami dla kazdego issue,
- `05-delivery/cycle-2/` - folder z planem drugiego cyklu MVP i rozbiciem discovery flow na dwa wieksze issues.

## Zalozenie projektu

Projekt ma byc sensowna aplikacja studencka:

- z animowanym ekranem startowym,
- z kilkoma ekranami,
- z requestami do zewnetrznego API,
- bez koniecznosci budowy osobnego backendu.
